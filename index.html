Коротка інформація про GNU/Linux

GNU/Linux — це операційна система з відкритим кодом, яку можна безкоштовно використовувати та змінювати. Вона базується на ядрі Linux, розробленому Лінусом Торвальдсом у 1991 році. Зараз існує велика кількість дистрибутивів GNU/Linux для різних цілей.

Найбільш популярні дистрибутиви

  • Ubuntu
  • Debian
  • Fedora
  • Arch Linux
  • Linux Mint
  • CentOS

Команди для роботи з Linux CLI

  • pwd – вивести поточну директорію
  • echo – вивести повідомлення
  • ls – вивести вміст директорії
  • cd – змінити директорію
  • touch – створити файл
  • mkdir – створити директорію
  • cp – копіювати файли
  • mv – перемістити або перейменувати файл
  • rm – видалити файл або директорію
  • history – вивести історію команд

BASH-скрипт авторизації


#!/bin/bash
echo "Введіть ім'я користувача:"
read username
echo "Введіть пароль:"
read -s password

if [ "$username" == "admin" ] && [ "$password" == "1234" ]; then
    echo "Авторизація успішна!"
else
    echo "Невірні дані!"
fi
